# Mapple-leaf-mess-management

MERN stack project: attendance, mess management, admin dashboard.

## Local setup

- **Backend:** `cd backend` → create `.env` with `MONGODB_URI`, `JWT_SECRET`, `JWT_EXPIRES_IN`, `ADMIN_PASSWORD` → `npm install` → `node server.js`
- **Frontend:** `cd frontend` → `npm install` → `npm run dev`

Do not commit `.env`; set `MONGODB_URI`, `JWT_SECRET`, and `ADMIN_PASSWORD` locally.
